# Raspberry Pi Pico

## Flash

Flash partitions

```
|------------------------------------------------|
|      A         | B |     C     |       D       |
|----------------|---|-----------|---------------|
|      1024K     |64K|  1472K    |     1536K     |
|------------------------------------------------|
                 |----------- flash.c -----------|

|------ 1MB -----|------------- 3MB -------------|
|---------------------- 4MB ---------------------|
```

- A : Binary (firmware)
- B : Storage (key-value database)
- C : User program (js)
- D : File system (lfs)
  (Total : 4MB)
